# ListenToThis

Next.js + TypeScript + Tailwind starter project.

## Kurulum

```bash
pnpm i   # veya npm i / yarn
pnpm dev # veya npm run dev
```

## Özellikler
- Next.js App Router
- TypeScript strict
- Tailwind CSS
- Basit parça listesi / arama / ekleme

## Proje Yapısı
```
app/
  page.tsx         # Ana sayfa
  layout.tsx       # Root layout
  globals.css      # Tailwind ve stiller
components/
  Header.tsx
  TrackCard.tsx
public/
  vercel.svg
```
